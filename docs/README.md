# Cot Guide

This directory contains the guide pages for the Cot web framework. The guide is built using Markdown files, which are
rendered into HTML by the website engine located in the [cot-site repository](https://github.com/cot-rs/cot-site).

## Building the website

To build the guide with the latest changes, you should go to the [docs-site](../docs-site/) directory and follow the
instructions from the [README.md](../docs-site/README.md) file. The changes will be reflected in the website as the "
master" version of Cot.
